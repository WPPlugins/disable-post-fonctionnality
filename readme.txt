=== Plugin Name ===
Tags: post, disable
Requires at least: 3.0.0
Tested up to: 3.3.1
Stable tag: 1.0


== Description ==

Disabled (hide) the Post functionnality of Wordpress.
Disabled "Post" in the admin menu et on the "Right Now" widget in the dashboard


== Installation ==

1. Upload `disabled-post-funcitonnality` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
